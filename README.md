# Customer Segmentation Project

## Objective
Segment customers based on purchasing behavior.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Method
K-Means Clustering

## Features
- Customer grouping
- Purchase pattern analysis
- Customer behavior analysis
- Visualization of customer segments

## Outputs
- customer_segments.csv
- customer_segments.png

## Conclusion
Customers were segmented into three groups based on Sales, Profit and Quantity purchased.
